# Viral Formula Library

## 一、标题学

### 7 大公式 + 中文示例 + 适用平台

| 公式 | 机制(心理学) | 中文示例 | 最适合平台 |
|------|---------------|----------|------------|
| 数字型 | 降低不确定性 | "5个让月薪 3000 也能精致的小习惯" | 小红书/公众号 |
| 否定避雷 | 提高具象画面 | "千万不要在这家店买羽绒服" | 抖音/小红书 |
| 反差对比 | 制造意外 | "从月薪 3000 到 3 万,我只做对了这 3 件事" | 抖音/B站 |
| 时效紧迫 | 制造稀缺 | "2026 年必入手的 10 款好物(月底涨价)" | 小红书/微博 |
| 痛点共鸣 | 群体认同 | "打工人都懂的早高峰崩溃瞬间" | 视频号/抖音 |
| 好奇缺口 | 信息不对称 | "为什么星巴克的服务员永远叫不出我名字?" | B站/公众号 |
| 人设口吻 | 信任迁移 | "作为 8 年程序员,我只推荐这 3 款键盘" | 小红书/B站 |

### 标题打分自检(0-10)

每个标题用以下维度自检,总分 ≥ 35 才发布:
- **具体度**: 越具体越好(数字/品牌/场景)
- **480ms 可读**: 移动端 0.5 秒能否抓住核心信息
- **情感强度**: 是否有明确的好奇/恐惧/愉悦
- **可信度**: 是否有"夸大"嫌疑
- **平台适配**: 是否符合该平台用户期待
- **长度**: 小红书 ≤20 字 / 公众号可长 / 抖音可短

### 学术依据

- Loewenstein 1994: 好奇心 = 信息缺口
- Scientific Reports 2025: "刚刚好"的具体度才是关键
- VidIQ Shorts 研究: 好钩子让 70-90% 用户继续看

## 二、封面学

### 封面公式

```
封面 = 主文字(≤5个词) + 强对比颜色 + 视觉焦点
```

### 5 种主要风格

1. **高对比红黑**: 强冲击(适合知识/数码/避雷类)
2. **温和粉色**: 治愈感(适合美妆/生活方式)
3. **极简留白**: 高级感(适合深度内容)
4. **人脸夸张**: 情绪传染(适合教育/反应类)
5. **产品特写**: 信息密度(适合开箱/测评)

### 设计自检

- 移动端最远阅读距离: 文字 ≥ 24pt
- 安全区: 不被推荐位遮挡
- 字体: 思源黑体粗体 / 站酷字体
- 关键信息: 1 主文字 + 1 副文字,不超 2 个层级
- 测试: 把封面缩到 200×200 像素能看清主信息吗?

### 行业经验数据

- TubeBuddy A/B 测试: 每变体 ≥ 500 impression 才有显著性
- 人脸 vs 无脸效果差异显著(品类强相关)
- 字数 ≤ 3 个词 vs ≥ 6 个词 CTR 差距 30-50%

## 三、节奏学

### 短视频(15-30 秒)

| 时间段 | 内容 |
|--------|------|
| 0-3 秒 | 钩子(承诺/反差/痛点)|
| 3-8 秒 | 立论(为什么相信)|
| 8-15 秒 | 高潮(最有信息密度的部分)|
| 15-25 秒 | 反转或升华 |
| 25-30 秒 | CTA(关注/点赞/评论)|

### 中长视频(5-10 分钟,B站/YouTube)

| 时间段 | 内容 |
|--------|------|
| 0-30 秒 | 强钩子(MrBeast 式 Jenga 开场)|
| 30s-2min | 立论与背景(快节奏)|
| 2-7 min | 主体内容(高密度,3-5 个分章节)|
| 7-9 min | 反转/升华/案例 |
| 9-10 min | 总结 + 三连请求 |

### 图文(小红书)

| 部分 | 要求 |
|------|------|
| 封面 | 1 主图 + 主文案 ≤8 字 |
| 副图 1-3 | 关键信息可视化 |
| 副图 4-6 | 步骤/案例展开 |
| 副图 7-8 | 总结/CTA |
| 正文 | 第一段 50 字内必须吸睛 |
| Tags | 5-8 个,2 个大词 + 3-5 个长尾词 |

## 四、爆款品类模板

### 美妆
- 测评对比 / 教程步骤 / 避雷清单 / 品牌种草 / 化妆品成分党
- 钩子套路: "素颜翻车救场" / "贵 vs 便宜对比"

### 数码
- 开箱 / 实测 / 横评 / 技巧 / 故障维修
- 钩子套路: "别买 / 真香反转" / "百元 vs 千元"

### 学习
- 干货清单 / 经验复盘 / 资源整理 / 打卡分享
- 钩子套路: "我踩坑你避坑" / "考研/考证速成"

### 生活方式
- Vlog / 居家好物 / 穿搭 / 旅行
- 钩子套路: "普通人 vs 小红书精致" / "100 元过一周"

### 知识
- 科普 / 深度解读 / 观点输出
- 钩子套路: "反常识但可核验" / "你以为是 A 但其实是 B"

### 搞笑
- 段子 / 反应视频 / 魔性循环 / 梗
- 钩子套路: "预期违背" / "情绪化夸张"

### 本地生活
- 探店 / 同城 / 性价比 / 避雷
- 钩子套路: "地图钉 + 价格" / "同城人均 X 元"

## 五、关键反爆款警示

避免:
- "标题党 + 内容空" → 平台降权
- "完美无瑕的人设" → 引发质疑
- "全员喜欢" → 没有立场反而平庸
- "节奏拖沓" → 完播率低
- "过度同质化" → 算法判定无价值
