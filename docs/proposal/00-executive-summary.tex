\section*{执行摘要}
\addcontentsline{toc}{section}{执行摘要}

\begin{insight}
\textbf{Ripple 2.0 是 KOC 的 Bloomberg Terminal——把资本与跨平台时差翻译成可发内容。}

不是 \textit{再做一个 Jasper 写文案}, 而是\textbf{ KOC 决策操作系统}: 一句话输入, 7 层架构 + 10 大原创能力流式输出可审决策, 每条结论附 \texttt{[source: url, ts]} 引用, 每个推荐可点 Why 看完整 Replay Graph 思考链。
\end{insight}

\subsection*{产品定位}

「KOC 追热点永远慢半拍, 等看到热搜时已经红海。Ripple 用 7 个真实数据源并行扫描刚开始升温的话题, 让你提前 7-14 天发现机会, 然后用 \textbf{Persona Vector + SimPredictor + Risk-Reward + Trend Chain + Campaign Planner} 做一场可审的内容战役决策。」

\subsection*{核心创新点}

\begin{enumerate}
    \item \textbf{7 层 OS 架构}: 不是流水线, 是可组合的原语 (40+ Tool + 5 层记忆)
    \item \textbf{Persona Vector}: 256 维向量 + EMA 增量更新 + Drift 熔断 + Git for Voice 版本控制
    \item \textbf{Replay Graph}: Merkle 链可审计 + 每一步决策可重放
    \item \textbf{SimPredictor}: 5 个虚拟受众 cohort 对多版本内容赛马
    \item \textbf{Risk-Reward Scorer}: 选题双头评分 + 推荐动作
    \item \textbf{Trend Chain}: 资本→国际→中文→大众 四阶段因果链推理
    \item \textbf{Cross-Platform Translator}: 一条源内容 → 6 平台改写
    \item \textbf{Cohort Affinity}: 分人群吸引力 + 破圈建议
    \item \textbf{Campaign Planner}: 7 天战役图 + 流量结构约束
    \item \textbf{Self-Critique Loop}: 仿 SELF-REFINE 自我批评闭环
\end{enumerate}

\subsection*{技术亮点}

\begin{itemize}
    \item \textbf{零硬编码 Mock}: 7 数据源全部真实 API 调用 (Polymarket / Manifold / HackerNews 直连; 微博/抖音/百度/B 站经第三方聚合)
    \item \textbf{真流式 SSE}: ChatGPT / Gemini Deep Research 级体验, 思考过程实时可视化
    \item \textbf{多层记忆}: 仿 Claude Code 5 层记忆 + 后台 Dream 巩固 + Markdown 指针索引
    \item \textbf{反 AI 投毒}: 引用强制 + 多源验证 + Merkle 链 + Drift 熔断
    \item \textbf{企业级安全}: BYOK 真实落地 (Argon2id + AES-256-GCM) + Audit Log + 权限分级
    \item \textbf{三端分发}: Web / MCP Skill / Tauri 桌面 (规划) / Docker
\end{itemize}
