# Ripple 3 分钟 Demo 录屏脚本(终版)

> **赛题要求**:MP4,≤3 分钟,≤500MB,带配音/字幕。
> **目标**:让评委在 3 分钟内记住 3 件事 — ① 我们做什么 ② 跟别人有什么不同 ③ 为什么能赢。

---

## 录屏前准备

| 项 | 内容 |
|---|---|
| **录屏工具** | macOS Screen Studio / OBS / QuickTime;Windows OBS / Bandicam |
| **分辨率** | 1920×1080(1080p) 或 2560×1440 (2K) |
| **帧率** | 30fps(降低体积,3min≈300MB) |
| **音频** | 用麦克风录口播(48kHz/16bit/单声道),或 AI 配音(MiniMax / 火山豆包) |
| **字幕** | 强制硬字幕(剪映 / FCPX / Premiere)。字号 ≥ 36px,白底黑字或黑底白字 |
| **背景音乐** | 极轻量(-25dB)或不要,避免抢话 |
| **环境** | 关掉所有通知;Demo 浏览器只开必要标签;桌面壁纸纯色 |
| **主持窗口** | 1 主窗口(Streamlit) + 1 辅助窗口(代码 / PDF) |

## 录制流程(执行步骤)

### Step 1:准备(D-2)
1. 整理 Demo 数据(确保"黄金"输入能产出令人惊艳的早期信号结果)
2. 浏览器只开:Streamlit Demo + 1 张架构图(备用)
3. 把代码窗口准备好(`agent_loop.py` 已打开到记忆系统部分)
4. 关闭所有通知:Mac 系统通知 / iMessage / 微信 / Slack

### Step 2:口播录制(D-1)
- 选项 A:本人对着稿子读 5-10 遍,挑最自然的一遍
- 选项 B:用 MiniMax / 火山豆包 TTS 合成,选"标准男声/女声",语速 1.0x

### Step 3:屏幕录制(D-1)
- macOS:**Screen Studio** 最佳(可后期加缩放/光标动画)
- 跨平台:**OBS Studio** 免费稳定
- 每个镜头单独录,后期拼接(更容易控制时长)

### Step 4:剪辑(D-1)
- **剪映专业版**(免费,自带 AI 字幕):导入素材 → 自动识别字幕 → 调整时间轴
- **DaVinci Resolve**(免费,专业):适合做转场动画
- **Final Cut Pro / Premiere**:有版权用就行

### Step 5:导出(D-0)
- 格式:MP4 (H.264 编码)
- 分辨率:1920×1080
- 帧率:30fps
- 码率:5-8 Mbps(3 分钟约 200-300MB)
- 音频:AAC 192kbps 单声道

### Step 6:上传备份
- 主提交:大赛指定平台
- 备份 1:B 站(unlisted 私密视频)
- 备份 2:腾讯微云
- 备份 3:本地双硬盘

## 风险与应对

| 风险 | 应对 |
|------|------|
| Demo 当场卡顿 | 提前录 3 个 Demo 镜头备份,任意一个用即可 |
| 配音口齿不清 | 用 AI 配音(MiniMax / 豆包 TTS) |
| 时长超 3 分钟 | 用 B 版本剪辑 |
| 文件超 500MB | 降码率到 5Mbps + 30fps,仍超就 720p |
| 字幕乱码 | 字幕用图片字幕(Picture-in-Picture)或转码为 .ass |
| 上传失败 | 多通道备份(B 站 / 微云 / 飞书 / 邮箱) |

## 自检清单(导出前)

- [ ] 视频时长 ≤ 180 秒
- [ ] 文件大小 ≤ 500MB
- [ ] 1080p / 30fps / H.264
- [ ] 全程有清晰口播或 AI 配音
- [ ] 全程有硬字幕(白底黑字 ≥ 36px)
- [ ] 开场 10 秒内出现产品名 + Logo
- [ ] 最后 10 秒内有联系方式 / Demo 链接
- [ ] 没有抖音 / 小红书等竞品 logo 出现(避免敏感)
- [ ] 没有未授权音乐(用 YouTube Audio Library 或站酷海洛免费音乐)
- [ ] 在 macOS / Windows / 手机至少 2 个设备预览验证
